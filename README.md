## 분석 항목

1. 교도 및 신도수 변화 (연령·성별)
2. 법회 출석 변화 (정례법회)
3. 출석 대상자의 주거지 현황 (시·군 상위 5)
4. 출석 대상자의 가족관계 (부부·부모자녀·조부모손자녀·형제)
5. 성장 추적에 따른 법회 출석 변화 (생애단계 잔존율)
6. 지역 인구 변화와 법회출석의 상관관계

## 참고

- 법회출석 = 정례법회(일반·청년·대학생·학생·어린이) 연 1회 이상 출석 교도수(중복 제외).
- 지구·급지 구분은 원기110년 교당현황 기준. 요양원·병원 등 기관교당은 지구·급지 분석에서 제외.
- §6 인구 데이터는 행정안전부 주민등록인구 근사치(2015 vs 2024).
- 원기 = 서기 − 1915 (원기100=2015, 원기110=2025).
